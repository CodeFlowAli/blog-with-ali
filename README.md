# CodeFlowAli Blog Website

A modern, responsive blog website built using HTML, CSS, and JavaScript.  
This project includes a custom admin panel for managing blog content.

---

## 🌐 Live Website

👉 "https://blogwithali.kesug.com/?i=1"

---

## 🚀 Features

- Fully responsive design (mobile + desktop)
- Clean and modern UI
- Blog post layout system
- Admin panel for managing content
- Lightweight and fast performance
- Built with pure HTML, CSS, JavaScript (no framework)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
